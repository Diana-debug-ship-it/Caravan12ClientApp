# Система автоматизации туристического агентства (Дипломный проект)

Комплексная клиент-серверная система, состоящая из двух независимых Android-приложений, взаимодействующих через единую облачную базу данных. Проект успешно защищен в качестве выпускной квалификационной работы (диплома) в 2023 году.

⚠️ **Текущий статус:** Проект успешно завершен и находится в архиве. В связи с закрытием учебной учетной записи конфигурация Firebase Cloud отключена. Проект сохранен для демонстрации навыков проектирования распределенных систем, ролевых моделей доступа и работы с NoSQL базами данных.

## 🏗️ Архитектура системы
Система разделена на два изолированных программных модуля, реализующих бизнес-логику для разных типов пользователей:

1. **Клиентское приложение:**
   * Регистрация и аутентификация пользователей.
   * Просмотр доступных туристических направлений.
   * Формирование и отправка заявок на подбор туров с указанием параметров.
   * Отслеживание статуса своих заявок в личном кабинете.

2. **Панель администратора:**
   * Авторизация сотрудников агентства.
   * Реализация CRM-системы: получение входящих заявок от клиентов в реальном времени.
   * Изменение статусов обработки заявок, фиксация заметок и контактных данных.

## 🛠️ Технологический стек и база данных
* **Язык разработки:** Kotlin
* **База данных и Backend:** Firebase Realtime Database / Firestore (NoSQL архитектура).
* **Аутентификация:** Firebase Authentication (управление сессиями пользователей).
* **Архитектурный паттерн:** MVVM в обоих модулях для изоляции UI от сетевых запросов.
